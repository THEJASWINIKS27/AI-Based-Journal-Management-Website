# AI Based Journal-Management-Website

The AI-Based Journal Management Website enhances the academic publishing process for article submission, peer review, and editorial management. 
AI-powered reviewer assignment, plagiarism detection, and manuscript summarization streamline the peer review process. 
Smart search and indexing system for easily discover relevant articles.

Academic journals are essential for sharing research findings, but traditional submission, peer review, and publication workflows face inefficiencies. 
This project presents the development of a Python-based web journal management system for material science research, enabling authors to submit papers, reviewers to assess them, and editors to manage the process seamlessly.
The system uses Django for the backend, SQLite 3 for efficient data management, and Role-Based Access Control (RBAC) with default Django authentication for secure access. 
AI-powered tools assist in reviewer assignment, automate workflows, improving efficiency and scalability for future growth.
